 See MPLS Pseudowire Types registry.

   Description

 MPLS pseudowire types must not be directly added to the "iana-pseudowire-types"
 YANG module. They must instead be added to the "MPLS Pseudowire Types"
 registry at [IANA registry pwe3-parameters].

 Module: the initial module file will be posted upon [RFC-ietf-opsawg-l2nm-19]'s publication as an RFC.
