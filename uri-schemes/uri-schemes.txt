Uniform Resource Identifier (URI) Schemes

   Expert(s)

 Registry: Mark Baker (primary), Murray Kucherawy (backup).
 CRI Scheme Number field: Unassigned.

   Reference
   [RFC7595][RFC Errata 4420][RFC8615][RFC-ietf-core-href-29]

   Note

 Requests for permanent registration must be preceded by mailing list
 review, per Section 7.2 of [RFC7595].


   Note

 Schemes for which the primary reference is an Internet-Draft are
 generally considered to be provisional until the I-D is approved by
 the IETF for RFC publication, even when the specification requests
 permanent registration.


   Note

 The CRI Scheme Number column registers numeric identifiers for the
 URI Schemes registered. Registrants for the Uniform Resource
 Identifier (URI) Schemes Registry may indicate that there are special
 requirements on the CRI scheme number to be assigned for the new URI
 Scheme.

 If that is not the case, IANA will assign a value autonomously. If
 there is a special requirement, the value will be allocated via
 Expert Review by the Designated Expert for the "CRI Scheme Number"
 column.

 Registrants that want to indicate special requirements on a CRI
 Scheme Number for a new URI-Scheme assignment are encouraged to
 notify the core-parameters@ietf.org mailing list of these
 requirements early. For more information, see [RFC-ietf-core-href-29].
