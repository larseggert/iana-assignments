   Note

 Formatting requirements for identifiers:

 * 1 character minimum length
 * 8 characters maximum length
 * Case-insensitive (values will be registered and published using
   upper-case characters)
 * String MUST contain only letters or digits as defined by XML 1.0.
   XML letter and digit characters are defined here:
   [http://www.w3.org/TR/2000/REC-xml-20001006#CharClasses]
