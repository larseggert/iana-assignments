   Registration Procedure(s)

 Standards Action

   Reference
           [RFC5838][RFC6969]

   Note

 These values for instance ID are applicable when [RFC5838] is
 used to support multiple address families. The instance ID field
 may be used for applications other than the support of multiple
 address families.

OSPFv3 Extended-LSA Sub-TLVs

   Reference
           [RFC8362]

   Note

 The column for the Applicability to L2 Bundle Member sub-TLV (L2BM)
 may be marked as follows:

 Y - sub-TLV MAY appear in L2 Bundle Member sub-TLV
 N - sub-TLV MUST NOT appear in L2 Bundle Member sub-TLV
 X - sub-TLV is not a Router Link sub-TLV; it MUST NOT appear
     in L2 Bundle Member sub-TLV
