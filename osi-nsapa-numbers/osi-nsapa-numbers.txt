   Note

 OSI Network Service Access Point Addresses include an Authority and
 Format Identifier (AFI). The AFI values 34 and 35 have been assigned
 to the IANA [ITU-T Recommendation X.213, X-Series Recommendations, Data
 Networks and Open Systems Communications, October, 2001.]. Within this format a two-octet Internet Code Point
 (ICP) field is available for assignment by the IANA. All ICP values
 not listed below are reserved.
