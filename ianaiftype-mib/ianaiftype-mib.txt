   Internet Assigned Numbers Authority

                                                                    IANAifType-MIB

   Created
           1994-01-31

   Registration Procedure(s)

 This is one of the available formats of the Interface Types (ifType) and Tunnel Types (tunnelType) registries.

   Expert(s)

 Dave Thaler (primary), Dan Romascanu (secondary)

   Description

 This MIB module defines the IANAifType Textual
 Convention, and thus the enumerated values of
 the ifType object defined in MIB-II's ifTable.
