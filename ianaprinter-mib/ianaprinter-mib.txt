   Description

 This MIB module defines a set of printing-related
 TEXTUAL-CONVENTIONs for use in Printer MIB [RFC3805],
 Finisher MIB [RFC3806], and other MIBs which need to
 specify printing mechanism details.

 Any additions or changes to the contents of this MIB
 module require either publication of an RFC, or
 Designated Expert Review as defined in [RFC8126],
 Guidelines for Writing an IANA Considerations Section
 in RFCs.  The Designated Expert will be selected by
 the IESG Area Director(s) of the Applications Area.
