Language Subtag Registry

   Registration Procedure(s)

 Expert Review

   Expert(s)

 Michael Everson (primary), Doug Ewell (secondary)

   Reference
   [RFC5645][RFC5646]

   Note

 Registration requests should be submitted to the
 [ietf-languages@ietf.org] mailing list. For more information,
 see [RFC5646], Section 3.5.


   Note

 The format of the registry is explained in Section 3 and
 Section 3.1 of [RFC5646].

 When referring to language tags in specifications, please refer to
 [RFC5646].


   Registry File


 [https://www.iana.org/assignments/language-subtag-registry]

 Language subtag registration forms are available in the

Language Tag Extensions

   Registration Procedure(s)

 IETF Review or IESG Approval

   Reference
   [RFC5646]

   Note

 While registration requires IESG approval or an IETF Stream RFC, the
 maintaining authority for an extension can submit changes to contact
 information and/or URLs directly to IANA, as described in [RFC5646],
 Section 5.2.


   Registry File


 [https://www.iana.org/assignments/language-tag-extensions-registry]
