   Note

 ATTENTION:  This registry has recently been combined with the Port
 Numbers Registry according to the recently approved document
 [RFC6335].
 For the new version of the registry, please see:
 [http://www.iana.org/assignments/service-names-port-numbers].
 This URL will soon redirect to the new registry and the old registry format
 will be removed.

 These are the Official Protocol Names as they appear in the Domain
 Name System WKS records and the NIC Host Table.  Their use is
 described in [RFC952] and will be combined into the port number
 registry per [RFC6335] when approved.

 A service name may be up to 15 characters taken from the set of
 letters, digits, and the special character hyphen.  It must start
 with a letter, and must consist of a combination of letters and digits.
 It must not start or end with a hyphen.
