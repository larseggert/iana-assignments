   Registration Procedure(s)

 See the TLS Application-Layer Protocol Negotiation (ALPN) Protocol IDs registry.

   Description

 New values must not be directly added to the "iana-http-versions" YANG module.
 They must instead be added to the "TLS Application-Layer Protocol Negotiation (ALPN) Protocol IDs"
 registry at [IANA registry tls-extensiontype-values].
