This registration tracks a change made to ISO 639-3 effective
2023-01-20, splitting the code element 'zua' for Zeem into five new code
elements, 'cxh' (Cha'ari), 'dsk' (Dokshi), 'dyr' (Dyarim), 'tvi'
(Tulai), and 'zem' (a new identifier for Zeem proper).

For more information on the ISO 639-3 change, refer to:
https://iso639-3.sil.org/request/2022-030
