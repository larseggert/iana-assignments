This registration tracks a change made to ISO 639-3 effective
2022-01-20, splitting the code element 'cug' for Chungmboko (Cung) into
two new code elements, 'bpc' (Mbuk) and 'cnq' (Chung).

For more information on the ISO 639-3 change, refer to
https://iso639-3.sil.org/sites/iso639-3/files/reports/639-3_ChangeRequests_2021_Summary.pdf
