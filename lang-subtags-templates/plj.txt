This registration tracks a change made to ISO 639-3 effective
2023-01-20, splitting the code element 'plj' for Polci into four new
code elements, 'nzr' (Dir-Nyamzak-Mbarimi), 'pze' (Pesse), 'uly' (Buli),
and 'zlu' (Zul).
