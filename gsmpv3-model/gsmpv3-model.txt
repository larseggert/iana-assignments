Model Type Names

   Reference

   Note

 Model Type Name values 0-1 are part of the GSMPv3 base
 protocol and are allocated through an IETF Review action.
 Model Type Name values 2-200 must be documented in an RFC
 or other permanent readily available reference.
 Model Type Name values in the range 201-255 are reserved
 for vendor private extensions and are the responsibility
 of individual vendors.
