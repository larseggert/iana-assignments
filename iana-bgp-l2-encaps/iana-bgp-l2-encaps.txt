   Registration Procedure(s)

 See BGP Layer 2 Encapsulation Types registry.

   Description

 BGP Layer 2 encapsulation types must not be directly added to the "iana-bgp-l2-encaps"
 YANG module. They must instead be added to the "BGP Layer 2 Encapsulation Types"
 registry at [IANA registry bgp-parameters].

 Module: the initial module file will be posted upon [RFC-ietf-opsawg-l2nm-19]'s publication as an RFC.
