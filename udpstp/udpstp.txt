   Reference
   [RFC9946]

   Note

 The algorithm identifier is a capitalized alphabetic UTF-8 value (A-Z),
 specified by the corresponding incremental numeric.

   A-Y   IETF Review
   Z     Reserved

   Value (Numeric)      Description       Reference
   A(n/a)          Not used              [RFC9946]

   C(1)            Rate algorithm Type C [TR-471]
