   Registration Procedure(s)

 See the Public Key Algorithm Names registry.

   Description

 New values must not be directly added to the "iana-ssh-public-key-algs"
 YANG module. They must instead be added to the "Public Key Algorithm Names"
 registry in the "Secure Shell (SSH) Protocol Parameters" registry group
 [IANA registry ssh-parameters].
